**3. Etiquetas multimedia**
    **Class#11**
        Tipos de imágenes
            Lossy vs Lossless:
                * Lossless(Sin pérdida), calidad.
                    -> Más pequeñas
                    -> Carga más rapida
                    -> Ejemplos: gif, PNG ...
                * Lossy(Con pérdida), calidad.
                    -> Más grandes
                    -> Carga menos rapida
                    -> Ejemplos: JPG / JPEG, SVG-Vector
    **Class#12**
